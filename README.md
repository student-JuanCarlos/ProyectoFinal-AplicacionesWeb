# 🛒 Sistema de Gestión: Logística, Inventario y Ventas
> Solución integral para el control de stock, gestión de sesiones y automatización de reportes.

![C#](https://img.shields.io/badge/C%23-512BD4?style=for-the-badge&logo=c-sharp&logoColor=white) ![SQL Server](https://img.shields.io/badge/SQL%20Server-0078D4?style=for-the-badge&logo=microsoft-sql-server&logoColor=white) ![n8n](https://img.shields.io/badge/n8n-FF6C37?style=for-the-badge&logo=n8n&logoColor=white)

---

### 🔑 Gestión de Accesos
El sistema implementa un control de seguridad basado en roles (RBAC):

| Rol | Alcance |
| :--- | :--- |
| **SuperUser** | Acceso total al sistema y gestión de usuarios/sesiones. |
| **Administrador** | Gestión de logística y ventas (sin acceso a usuarios). |
| **Trabajador** | Visualización de productos y generación de ventas. |

---

### 🤖 Automatización con n8n
El sistema no es solo una base de datos; incluye un **ChatBot inteligente** que permite:
* **Reportes:** Generación automática de PDFs.
* **Consultas:** Respuesta en tiempo real sobre stock y movimientos.
* **Integración:** Flujos de trabajo automatizados para consultas rápidas.

---

### ✅ Características
- [x] **Control de Usuarios:** Registro y desactivación de cuentas en tiempo real.
- [x] **Gestión de Stock:** Seguimiento exacto de entradas y salidas.
- [x] **Módulo de Ventas:** Interfaz optimizada para el rol de trabajador.
- [x] **Inteligencia de Datos:** ChatBot para dudas de inventario.
